<p>Para probar la aplicación de descuento se pueden utilizar los siguientes codigos</p>
<ul>

  <li><b>desc25</b>: Aplica un 25% de descuento en el total</li>
  <li><b>desc50</b>: Aplica un 50% de descuento en el total</li>
</ul>
